## Análisis de imagen

Para analizar imagen podríamos recurrir a varias estrategias. 

Podríamos analizar pixel por pixel y podríamos estimar qué pixeles cambian y en que zonas. 

También podríamos usar openCV. 

El día de hoy vamos a usar el paquete [mediapipe task visions ](https://www.npmjs.com/package/@mediapipe/tasks-vision) para detectar objetos y establecer acciones a partir de esas detecciones. 

Podemos explorar un [detector de persona](https://emilioocelotl.github.io/detector-de-persona/) en el navegador.
